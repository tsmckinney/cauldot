# Cauldot Engine authors

Cauldot is a fork of Godot developed by lots of friendly people who
like contributing code, reporting bugs, requesting new features, etc.,
across plenty of repositories.

We could never truly list every single one in just one markdown document, so
this file aims to showcase those who made the most important changes to
Cauldot.

The names of Cauldot developers will be formatted like this: 
FirstName LastName (githubUsername, pronouns) or githubUsername 
(pronouns), except without the comma or the linebreaks if any appear on 
the device you're viewing on.

This list will again remain curated and use a minimum of 11 commits.
